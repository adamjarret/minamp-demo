This is a live demo of the minamp theme for Hugo.

The minamp theme is intended to be a blank canvas for AMP sites. It is focused on versatility and reducing the pain of loading styles and AMP component scripts from different scopes.

This example site features a sidebar and photo gallery as an illustration of how the theme can be customized.
